# Martin Bot (Version Iron)

A simple terminal bot based of Martin from Cortis cuz c.ai is for incels, nerds and people with internet 
Thanks to Niyo for the ragebait

---

## Features

- Persistent username memory (remembers your name for future chats)  
- Mood-event detection: recognizes sentences like "I am feeling sad/happy/angry/confident" and replies accordingly  
- Keyword-based replies with priorities  
- Suggest songs based on your emotion (type: suggest me a song i am feeling ___.) 

---

## How to Use (Windows Users)

Martin Bot runs **on Windows using Git Bash** — you do **not** need Python or any other software.

### Step 1: Install Git Bash
1. Go to [Git for Windows](https://git-scm.com/download/win)  
2. Download and run the installer  
3. Follow the instructions (default settings are fine)  
4. Once installed, you can open **Git Bash** from the Start menu  

---

### Step 2: Download the Martin Bot
You can either:

**Option A: Clone via Git (recommended)**  
1. Open Git Bash  
2. Navigate to the folder where you want to store the bot, e.g., your Desktop:  
   ```bash
   cd ~/Desktop
   ```  
3. Clone the repository 
   ```bash
   git clone <https://github.com/Rohit-dev-Unity/MartinBot.git>
   cd MartinBot
   ```  

**Option B: Download ZIP**  
1. Go to the GitHub repository page <https://github.com/Rohit-dev-Unity/MartinBot.git>
2. Click **Code → Download ZIP**  
3. Extract the ZIP to a folder, e.g., `MartinBot`  
4. Open Git Bash in that folder  

---

### Step 3: Run Martin Bot
1. Make sure the script is executable (first time only):  

   chmod +x martin.sh

2. Start chatting:  

   ./martin.sh

3. Type `exit` to quit the chat
